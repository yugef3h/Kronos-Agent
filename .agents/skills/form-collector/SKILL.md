# 表单收集 Skill
## 用途
多轮对话收集用户基础信息（姓名/年龄/职业），自动校验合法性，用户闲聊/乱输入会被拦截，不打断流程。

## 触发条件
用户需要登记信息、个人资料、简单表单录入时使用。

## 执行步骤
1. 初始化会话，要求用户输入姓名
2. 调用 LLM 校验姓名合法性，合法则进入年龄收集
3. 校验年龄（1-120），合法则进入职业收集
4. 校验职业，完成后汇总信息
5. 任意步骤非法，提示重输，不切换状态
